# cctv-zhongchao-archive

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态 HTML 文件形式存储，不依赖于特定域名或具体网站，旨在提供一种轻量、可复用的内容发布方式。

## 目录说明

```
/
├── pages/          # 存放各独立 HTML 页面文件
├── assets/         # 页面引用的静态资源（如 CSS、JS、图片等）
├── index.html      # 仓库总入口页面（可选）
└── README.md       # 本文件
```

- `pages/` 目录下的每个子目录或文件对应一个独立归档单元。
- `assets/` 目录用于存放页面所需的公共资源，便于统一管理。
- 仓库根目录的 `index.html` 可作为导航或说明页面（如存在）。

## 页面归档说明

- 每个 HTML 页面均为独立完整文件，可直接在浏览器中打开查看。
- 页面内容不涉及具体域名或网站，仅以纯前端形式呈现归档信息。
- 页面间无强制依赖关系，可单独使用或通过索引页面进行跳转。
- 所有页面均遵循通用 HTML 规范，兼容主流浏览器。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面时请保持目录结构清晰。
- 提交前请确保 HTML 文件无外部不可达资源链接，避免引用失效。
- 如对页面内容或结构有疑问，欢迎通过 Issues 进行讨论。
- 本仓库仅用于技术归档目的，不提供任何形式的服务承诺或内容保证。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 目录下的任意 HTML 文件。
3. 也可通过根目录 `index.html` 浏览全部页面索引。
